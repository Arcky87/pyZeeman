Команда для удаления (выполнять осторожно!)

```bash

НЕ ВЫПОЛНЯТЬ без резервной копии!

Сначала создайте бранч в git или резервную копию

Удаление устаревших модулей

rm simple_tracer.py rm tracer.py rm extractor.py rm extract_slicer.py rm extract_and_rebin.py rm LA_Cosmic.py rm LA_image.py rm list_cosmo_cleaner.py rm sl_remover.py rm thar_rebin.py rm blz_correct.py rm surf_fit.py rm load_trace.py rm .extractor.py.swp

Рекомендации перед удалением

  1. Создайте резервную копию или git commit текущего состояния
  2. Создайте новую ветку в git: git checkout -b cleanup-deprecated
  3. Запустите тесты (если есть) после удаления
  4. Проверьте работу пайплайна на тестовых данных
  5. Сделайте commit изменений с подробным описанием
